# README

## GrowthCircle by Ria Varghese

### Ruby version 
3.3.6

### Rails version
7.2.2

### Database
MySQL

## Setup
### Setup dependencies 
```bash
bundle install
```

### Setup database
```bash
bin/rails db:create

bin/rails db:migrate

bin/rails db:seed
```



### Startup
```bash
./bin/dev
```
